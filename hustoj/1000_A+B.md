## 题目地址（#1000_A+B）

http://www.hustoj.org/problem/1000

## 题目描述

```cpp
请补充完成下面的函数，返回两个参数的和。

int plus_2_int(int a,int b){

}

输入：1 2
输出：3
```

## 解法

1. 常规解法

一个开胃菜，方便解题者了解oj的使用方式以及代码提交模板。

```cpp
#include <iostream>
using namespace std;
int main(){
    int a,b;
    while(cin >> a >> b)
        cout << a+b << endl;
	return 0;
}
```
