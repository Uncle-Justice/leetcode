## 题目地址(0136.只出现一次的数字)

https://leetcode-cn.com/problems/single-number/

## 题目描述

## 关键词：位运算

## 解法

1. 异或

空间O(n)的方法好想，但是异或的方法还真的想不到。。。


```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int res = 0;
        for (auto& x: nums) {
            res = res ^ x;
        }
        return res;
    }
};
```
